# PWA Icons

Bu dizine aşağıdaki icon dosyalarını eklemelisiniz:

- `icon-192.png` - 192x192 piksel PNG icon (emerald (#059669) arka plan, beyaz belge simgesi)
- `icon-512.png` - 512x512 piksel PNG icon (emerald (#059669) arka plan, beyaz belge simgesi)

Şu an için `manifest.json` bu dosyalara referans veriyor. Icon'ları oluşturmak için `icon.svg` dosyasını kullanabilir veya online araçlar kullanabilirsiniz:

- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator

Alternatif olarak, icon.svg dosyasını 192x192 ve 512x512 boyutlarında PNG'ye dönüştürebilirsiniz.
